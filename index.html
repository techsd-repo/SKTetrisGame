Tetris, with a twist

Instead of making a normal tetris game like every other one on the app store, I wanted to make something abstract.. But, at the same time it was Tetris not some other game.

You can get the code for this on github, with the buttons above, or clone it using this link:

$ git clone https://github.com/techsd-repo/SKTetrisGame.git

Feel free to make suggestions and contributions to make this game better

Why Swift, not Objective-C

I wasn't really on the whole Swift train at first, but then I realized that the language was a lot better in my opinion. Swift just naturally came to me, I didn't go and learn it online.
